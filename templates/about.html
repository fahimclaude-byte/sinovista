🎯 Project Mission

SinoVista is an AI-powered web platform that helps travelers explore Chinese cities through real-time weather intelligence, personalized recommendations, and smart safety alerts. The name combines "Sino" (referring to China) with "Vista" (a beautiful view), reflecting our goal: to give every traveler a clear, beautiful view into China's diverse cities.

🏗️ System Architecture

SinoVista follows a clean three-layer Object-Oriented architecture:

  • Presentation Layer: Flask web framework with Jinja2 templates, custom CSS, and vanilla JavaScript for interactivity.
  • Business Logic Layer: Four core OOP classes — CityManager (data management), WeatherService (API integration), AIRecommender (AI logic), WeatherAlertSystem (rule-based reasoning).
  • Data Layer: JSON city database + real-time Open-Meteo Weather API + LLM API (DeepSeek/OpenAI ready).

🧠 Advanced Features

  • Smart AI Recommendations: AI analyzes the 5-day weather forecast and your preferences to build a custom itinerary.
  • Interactive China Map: Click any city on the map to instantly view its details and weather.
  • Weather Comparison Charts: Compare up to 3 cities side-by-side with beautiful visualizations.
  • Favorites System: Save your favorite cities for quick access (uses localStorage).
  • China Knowledge Quiz: Interactive 10-question quiz with scoring system.
  • Weather-Aware Planning: Outdoor activities for sunny days, indoor alternatives for rainy ones.
  • Smart Alerts: Rule-based AI scans weather for storms, extreme temperatures, wind, fog, and more.
  • Multilingual Support: Available in English, বাংলা (Bengali), and 中文 (Chinese).
  • Dark/Light Theme: Adaptive UI for comfortable viewing day or night.
  • AI Chat Assistant: Ask any travel question in natural language.
